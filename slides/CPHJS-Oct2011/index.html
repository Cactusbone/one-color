A little domain knowledge

Displays

RGB

HSV

  • Hue: 360 degrees pure color
  • Saturation: colorfullness
  • Value: amount of light
  • More intuitive than RGB

HSL

  • Lightness vs value

Color representations

  • Hex: "#123456"
  • RGB: "rgb(100, 100, 100)"
  • RGBA: "rgba(100, 100, 100, 0.5)
  • HSL: "hsl(100, 100, 100)"
  • HSLA: "hsla(100, 100, 100, 0.5)
